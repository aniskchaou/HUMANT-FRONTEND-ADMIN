What is HUMANT?

HUMANT is a Human Resource Management System designed to centralize employee lifecycle, recruitment, payroll workflows, attendance, leave, communication, and compliance records in a single operational workspace.

The platform is intended for organizations that require traceable workflows, role-aware access, and measurable service quality across people operations.

Business Scope
  • Employee lifecycle management from onboarding to exit
  • Recruitment cycle from job creation to offer management
  • Compensation workflows including salary, payslip, loan, and expenses
  • Operational governance through warning, complaint, and termination records
  • People growth through training, performance, awards, and events
Target Outcomes
  • Reduce manual handoff errors between HR and Payroll teams
  • Improve processing times for leave, hiring, and approvals
  • Ensure repeatable policy execution with transparent audit trails
  • Provide managers and leadership better operational visibility
  • Increase consistency through master-data-driven workflows
Reference Architecture
Frontend Layer

Angular web workspace with module-driven navigation, forms, lists, and dashboards for operational teams.

Application Layer

Spring Boot services exposing secured APIs, enforcing business rules, and coordinating transactional updates.

Data Layer

PostgreSQL persistence model with startup initialization support and schema evolution through controlled release changes.

Technology and Governance Notes
Core Stack
  • Frontend: Angular 11, Bootstrap, RxJS
  • Backend: Spring Boot 2.5, Spring Data JPA, Spring Security
  • Runtime: Java 11, Node.js
  • Data: PostgreSQL (primary), MySQL legacy compatibility
Control Model
  • Role-based route visibility and guarded access flow
  • Sensitive area protections for salary and document domains
  • Recommended maker-checker pattern for financial approvals
  • Audit-oriented history retention for lifecycle-critical actions
